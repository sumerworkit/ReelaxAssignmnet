# Reelax Figma Assignment

A responsive React implementation of the Reelax checkout and billing interface from the supplied Figma design.

## Tech Stack

- React JS
- Vite
- CSS Modules-style global CSS
- Lucide React icons

## Features

- Pixel-focused checkout layout based on the Figma reference
- Billing information form
- Order summary panel
- Wallet balance and coupon selection UI
- Responsive desktop, tablet, and mobile behavior
- Component-based architecture

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
